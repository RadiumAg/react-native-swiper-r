# react-native-swiper-r

Api

| 参数          | 说明                      | 类型                  | 默认值   |
| ------------- | ------------------------- | --------------------- | -------- |
| autoplay      | 是否自动切换              | boolean               | true     |
| cardSetting   | 卡片设置，详见下图        |                       |          |
| mode          | normal和cardSide,详见下图 | 'normal' \|'cardSide' | 'normal' |
| contentOffset | 初始位置                  | number                | 0        |
